#### 4. Parallel Lines (ID: f4dfd00f-06c1-4f9b-af90-72815d9828f4)
**Question:** Which of the following lines is NOT parallel to the line 3y+2x-17=0?
- **Old Answer:** B (9y+6x+17=0)
- **Correct Answer:** D (3y-2x+70=0)
- **Calculation:**
  - Original line: 3y = -2x + 17, slope = -2/3
  - Option B: 9y = -6x - 17, slope = -6/9 = -2/3 (parallel)
  - Option D: 3y = 2x - 70, slope = 2/3 (NOT parallel - different slope)
- **Status:** ✓ CORRECTED in database

#### 5. Derivative at a Point (ID: d2d09a5e-c4c6-41ae-b027-a826c9626d9c)
**Question:** If y=(1-2x)³, find the value of dy/dx at x=-1
- **Old Answer:** B (27)
- **Correct Answer:** D (-54)
- **Calculation:**
  - dy/dx = 3(1-2x)² × (-2) = -6(1-2x)²
  - At x=-1: -6(1-2(-1))² = -6(1+2)² = -6(9) = -54
- **Status:** ✓ CORRECTED in database

#### 6. Pentagon Interior Angles (ID: ece39890-ba18-42f7-97d2-22649ac67ce0)
**Question:** The interior angles of a pentagon are (2x+5), (x+20), x, (3x-20) and (x+15). Find the value of x.
- **Old Answer:** A (80)
- **Correct Answer:** C (65)
- **Calculation:**
  - Sum of interior angles = (2x+5) + (x+20) + x + (3x-20) + (x+15) = 8x + 20
  - Pentagon sum = (5-2) × 180 = 540°
  - 8x + 20 = 540
  - 8x = 520
  - x = 65
- **Status:** ✓ CORRECTED in database

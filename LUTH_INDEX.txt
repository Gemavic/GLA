================================================================================
TROUBLESHOOTING
================================================================================

OCR Issues:
  - Poor text quality → Increase PDF resolution before OCR
  - Mixed characters → Use English-specific OCR settings
  - Spacing issues → Use post-processing script

Parsing Issues:
  - Questions not detected → Verify question number pattern
  - Options missing → Check option format (A), B), etc.
  - Subject wrong → Add more keywords to detection

Database Issues:
  - exam_body not found → Create LUTH exam body first
  - Foreign key error → Verify subject IDs exist
  - Duplicate error → Remove duplicate questions
